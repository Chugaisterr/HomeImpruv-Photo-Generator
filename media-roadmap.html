Rules

  • All lowercase, no spaces — use underscore _ as separator
  • Sequence is always 3 digits: 001, 012, 099, 100
  • Before/After pairs share the same sequence number
  • Output format always .jpg (convert webp/png/avif on rename)
  • No Cyrillic, special chars, or parentheses in filenames
  • Sub-niche (shower) is separate niche code, NOT a subfolder of bathroom

GPT-4o Mini Classifier Spec

Input per image

{
  "image": "<base64 jpeg, max 512px>",
  "current_folder": "HVAC/Hero Page",
  "current_filename": "istockphoto-xxx.jpg"
}

Prompt structure

1. Identify niche: hvac | bathroom | shower | flooring | siding | security
2. Identify use: hero | ba | project
3. Identify subtype based on use
4. Detect: has_text_overlay (bool)
5. Detect: has_person (bool)
6. Quality score 1-10
7. Return JSON only

Current Issues & Gaps
🔤
Inconsistent filenames — raw download names
Many files still have stock-site names (istockphoto-xxx, 360_F_xxx, pexels-xxx), UUID names, or Cyrillic characters. These block automation because you can't infer content type from filename.
📂
Folder naming not standardized
"Befor After" (HVAC), "Before-after" (Bathroom), "before after" (Flooring), "BeforeAfter" (Shower) — all mean the same thing. Scripts can't work without normalization first.
🔀
BA folder mixes 5 different sub-types
Together (both sides in 1 image), Before only, After only, Story/vertical, Process/during — all dumped in one BA folder. The model needs to know the sub-type to process correctly.
🖼️
Multiple formats: jpg + png + webp + avif
Mixed formats complicate batch processing. Target: all .jpg for consistency. Convert on rename.
🏠
Security niche has no BA or Project folders
Only Hero Page exists for Security. Either the niche doesn't have BA content yet, or it was never sorted. Check if BA photos need to be collected.
📹
Video files in public/ folder — unhandled
3 .mp4 files found in public/. No video processing workflow exists yet. Need to decide: process separately or ignore for now.
✅
HVAC folder has best structure — use as reference
HVAC has the clearest manual naming: "HVAC Before After Together 1.jpg", "HVAC Before 1.png" etc. This pattern guided the naming convention above.
